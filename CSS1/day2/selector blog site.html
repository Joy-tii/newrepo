The Ultimate Guide to CSS Selectors

CSS selectors are the backbone of styling web pages, allowing developers to target specific HTML elements and apply styles to them. This blog explores all major types of CSS selectors, their syntax, and examples of their use.

Universal Selector Class Selector Id Selector Group Selector Type Selector Descendent Selector Pseudo Class

Universal Selector (*)

The universal selector applies styles to all elements in a document.

Syntex:

* {
property: value;
}

examples:

* {
margin: 0;
padding: 0;
box-sizing: border-box;
}


Class Selector (.)

The class selector targets elements with a specific class attribute.

Syntex:

.classname {
property: value;
}

examples:

.button {
background-color: green;
color: white;
}


ID Selector (#)

The ID selector applies styles to a single element with a unique id.

Syntex:

#idname {
property: value;
}

examples:

#header {
font-size: 24px;
font-weight: bold;
}


Group selector

The group selector applies the same styles to multiple selectors.

Syntex:

selector1, selector2, selector3 {
property: value;
}

examples:

h1, h2, h3 {
font-family: Arial, sans-serif;
}


Type Selector

The type selector targets elements by their HTML tag name.

Syntex:

element {
property: value;
}

examples:

h1 {
color: blue;
}
p {
line-height: 1.5;
}


Descendant Selector (Space)

The descendant selector targets elements that are nested inside another element.

Syntex:

parent child {
property: value;
}

examples:

div p {
color: gray;
}


Pseudo-Classes

Pseudo-classes target elements based on their state or position.

Syntex:

selector:pseudo-class {
property: value;
}

examples:

a:hover {
color: red;
}
